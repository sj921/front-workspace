# Browser Object Model (BOM)

  • 웹 브라우저의 기능을 활용하기 위한 객체 (window)
  • DOM도 웹 브라우저 객체의 내부에 존재한다 (window.document)
  • window는 생략이 가능하다

# BOM 내부의 여러 객체들

  • window: 웹 브라우저 최상위 객체
  • window.screen: 모니터에 대한 정보
  • window.document: DOM. 현재 페이지의 HTML코드 구조
  • window.history: 웹 브라우저의 페이지 이동 내역
  • window.location: 현재 웹 브라우저의 위치에 대한 정보
  • window.navigator: 현재 웹 브라우저의 정보
