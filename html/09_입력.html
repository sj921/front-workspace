# input 태그를 사용해 다양한 입력을 받을 수 있다

Type Sample Description Usage
type="text" 일반 텍스트 입력칸 아이디, 글 제목
type="password" 비밀번호 입력 칸 비밀번호
type="radio"



여러가지 중 하나만 선택할 수 있는 버튼 성별, 연령대 등
type="checkbox"



같은 카테고리 내에서 여러가지를 동시에 선택할 수 있는 버튼 관심분야 등
type="date" 날짜를 입력받는 칸
<input>에는 이 외에도 많은 type들이 있다....
<textarea> 긴 내용의 텍스트를 입력받을 수 있는 태그 (input 태그 아님) 글 내용, 문의 내용
<select>, <option> 직책 : 여러 옵션 중 하나를 선택할 수 있음 주로 외래키 컬럼 (정해진 것 안에서 골라야 하는 경우)
