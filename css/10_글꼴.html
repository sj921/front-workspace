# font-family

  • font-family 속성으로 원하는 글꼴을 선택할 수 있다
  • 기본적으로는 설치되지 않은 글꼴은 사용할 수 없다
  • 글꼴을 사용할 수 없는 경우에 대비하여 여러가지 예비용 글꼴들을 설정해 둘 수 있다
  • 접속한 사용자에게 글꼴을 설치하게 하는 것보다는 CDN을 많이 활용한다
  • CDN : Content Delivery Network, 다른 사람이 인터넷에 올려놓고 배포하는 이미지, 글꼴 등의 리소스들을 의미한다
