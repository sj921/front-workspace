# Display 속성

  • 해당 요소가 어떻게 보여야 할지를 결정하는 속성
  • 요소가 보이는 방식에는 기본적으로는 block과 inline이 있다

# Block

  • 항상 한 줄을 모두 차지한다
  • 기본 너비는 화면 100%이다
  • padding과 margin을 적용할 수 있다
  • 박스 모델
  • ex: div, p, h, ul, li, ol, ... 등등

# Inline

  • 영역을 지정하더라도 새 줄에서 시작하지 않는다
  • 글 내용에서 특정 부분을 선택할 때 사용된다
  • padding과 margin을 올바르게 적용할 수 없다/li>
  • 글자 취급
  • ex: a, b, i, mark, ... 등등

  • padding과 margin을 적용할 수 있으면서 글 내용과 어우러질 수 있다
