  • in: 인치

# 상대 길이 단위들

  • em : 현재 요소의 font-size에 비례한 크기 설정
  • rem : 현재 페이지(html 태그)의 font-size에 비례한 크기 설정
  • vw : viewport width, 현재 브라우저 너비에 비례한 크기 설정
    1vw는 현재 전체 브라우저 viewport 너비의 1%를 의미함
  • vh : viewport height, 현재 브라우저 높이에 비례한 크기 설정
    1vh는 현재 브라우저 viewport 높이의 1%를 의미함
