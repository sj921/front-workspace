# Position

  • 포지션이 설정되지 않은 다른 요소들을 무시하면서 마음대로 배치할 수 있는 속성
  • static - default, 포지션이 설정되지 않은 상태
  • absolute - 포지션이 설정된 부모 요소 내부에서 절대적인 위치를 설정
  • relative - 원래 위치로부터 상대적인 위치를 설정
  • fixed - viewport에서 고정된 위치를 설정
  • sticky - 스크롤이 변할 때 고정될 위치를 설정
  • 포지션을 설정한 후에는 top, left, bottom, right 속성들로 해당 요소의 위치를 결정할 수 있다
