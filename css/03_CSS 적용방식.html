# 페이지에 스타일 시트를 적용하는 방법 3가지

1. inline CSS (줄 내부 CSS)
- HTML style attribute를 사용하는 방식
- CSS 적용 우선 순위가 가장 높다
2. internal CSS (파일 내부 CSS)
- 파일 내부의 style tag를 사용하는 방식
- 더 나중에 나온 스타일시트가 앞의 내용을 모두 덮어쓴다
3. external CSS (파일 외부 CSS)
- 외부의 .css 파일에서 스타일 시트를 가져와 사용하는 방식
- 외부의 .css 파일을 가져올 때 link tag를 사용한다
- 우선 순위는 internal CSS와 같다
