# Margin

  • 해당 요소와 다른 요소의 거리를 결정하는 여백
  • border 바깥쪽의 여백
  • bottom margin과 top margin이 만나는 곳에서는 둘 중 더 높은 margin만 적용된다
  • margin을 auto로 설정하면 가운데 정렬을 해준다

# Padding

  • border 안쪽의 여백
  • 요소 내부의 내용으로부터 border까지의 거리를 의미하는 여백

# HTML Box Model

HTML의 모든 요소는 박스 형태로 구성되어 있다.
박스를 구성하는 박스의 제일 바깥쪽 여백이 margin이고, border, padding, content 순으로 요소를 구성하고 있다

# Box Sizing Property

  • box-sizing:content-box는 width가 content-box의 크기를 의미한다 (기본 설정)
  • box-sizing:border-box는 width/height가 border-box의 크기를 의미한다
Content-box's width 500px / height 200px
(요소의 전체 크기 = width/height + padding + border)
Border-box's width 500px / height 200px
(요소의 전체 크기 = width/height)
