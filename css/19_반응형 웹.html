# 반응형 웹 사이트

  • viewport의 크기에 따라 모양이 달라지는 웹사이트
  • 각 크기에 알맞는 최적화된 화면을 제공할 수 있다
  • CSS의 media query를 사용한다
  • @media (미디어종류) and (원하는크기)

# 미디어 종류

  • all : 모든 타입
  • print : 출력 화면에서의 스타일 지정
  • screen : 컴퓨터, 태블릿, 스마트폰 등의 일반 기기
  • speech : 화면을 읽어주는 기기 등
